## 5. 风险预判与应对措施

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|---------|
| openpyxl 未安装导致导出失败 | 中 | 低 | 运行时提示安装命令，不影响核心功能 |
| 数据库文件损坏 | 低 | 高 | WAL 模式减少损坏概率，定期备份建议 |
| Tkinter 在不同 Windows 版本上渲染差异 | 低 | 低 | 使用标准 ttk 控件，颜色使用十六进制值 |
| 打包后 ECharts 路径问题 | 中 | 中 | 使用 `sys._MEIPASS` 机制定位资源文件 |
| 大数量记录导致 UI 卡顿 | 中 | 中 | 限制 500 条显示，后续可加分页 |
| Python 版本兼容性 | 低 | 中 | 使用 Python 3.10+ 语法特性（`type \| None`） |
