# 生产管理系统 — 桌面端应用设计文档（v2.0 业务重构）

> 版本 2.0 | 2026-07 | 电气行业专用

---

## 1. 需求变更说明

根据用户反馈，对以下三个核心模块进行业务重构：

| 模块 | 变更内容 | 说明 |
|------|---------|------|
| 📦 物料管理 | 去除单价，增加"编号+名称+版本"三维结构 | 一个物料可有多个版本，适应电气行业SKU管理 |
| 👷 工人管理 | 树形展示(班组→工人)，班组改为下拉选择 | 班组作为独立实体，工人从属于班组 |
| 🔧 工序管理 | 去除工人分配，物料/工序改为选择+搜索形式 | 物料显示"编号(名称)-版本"，支持关键字搜索 |

---

## 2. 功能模块清单（修改后）

### 2.1 物料管理模块

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| 编号 (code) | TEXT UNIQUE | 物料编码，手动输入 | TR-001, CONT-001, CB-001 |
| 名称 (name) | TEXT | 电气物料名称 | 变压器, 接触器, 断路器 |
| 版本 (version) | TEXT | 物料版本号 | V1.0, A型, 标准型 |

**说明**：
- 同一名称+版本不能重复（UNIQUE(name, version)）
- 一个物料（如"变压器"）可以有多个版本（V1.0, V2.0, 加强型）
- 物料命名参考电气行业标准：
  - 变压器 (Transformer)
  - 接触器 (Contactor)
  - 断路器 (Circuit Breaker)
  - 继电器 (Relay)
  - 电缆 (Cable)
  - 开关 (Switch)
  - 熔断器 (Fuse)
  - 电容器 (Capacitor)
  - 互感器 (Instrument Transformer)
  - 母线 (Busbar)
  - 配电箱 (Distribution Box)
  - 控制柜 (Control Cabinet)

### 2.2 工人管理模块

| 实体 | 说明 | 操作 |
|------|------|------|
| 班组 (Group) | 父级节点，如"装配组""测试组" | 添加/删除/重命名班组 |
| 工人 (Worker) | 子级节点，从属于某个班组 | 添加/编辑/删除，所属班组从下拉选择 |

**树形结构示例**：
```
┌─ 装配组
│  ├── 张三
│  ├── 李四
│  └── 王五
├─ 测试组
│  ├── 赵六
│  └── 孙七
└─ 包装组
   ├── 周八
   └── 吴九
```

### 2.3 工序管理模块

| 字段 | 输入方式 | 说明 |
|------|---------|------|
| 物料 | 可搜索下拉（Combobox） | 从物料库中选择，显示"编号(名称)-版本" |
| 工序名称 | 可搜索下拉+可输入 | 从已有工序名中选择或输入新名称 |
| 单价 | 数字输入 | 该工序的计件单价 |

**移除**：工人分配区域（原勾选分配工人的功能全部去掉）

## 4. 界面设计（修改部分）

### 4.1 物料管理对话框（重构）

```
┌─────────────────────────────────┐
│  管理物料                    - □ × │
├─────────────────────────────────┤
│  物料列表         （支持多版本）   │
│  ┌──────┬────────┬──────────┐   │
│  │ 编号  │ 名称   │ 版本     │   │
│  ├──────┼────────┼──────────┤   │
│  │TR-001│ 变压器 │ V1.0    │   │
│  │TR-002│ 变压器 │ V2.0    │   │
│  │CONT-01│接触器  │ A型     │   │
│  │CB-001│断路器  │ 标准型   │   │
│  │RE-001│继电器  │ V1.0    │   │
│  │ ...  │ ...    │ ...     │   │
│  └──────┴────────┴──────────┘   │
│                                   │
│  编号:[        ] 名称:[        ]  │
│  版本:[        ]                  │
│  [添加] [编辑] [删除]             │
└─────────────────────────────────┘
      尺寸: 600×450
```

**交互逻辑**：
- 添加：填写编号、名称、版本 → 验证唯一性 → 添加到列表
- 编辑：选中行 → 弹出编辑窗口 → 修改后保存
- 删除：选中行 → 确认 → 删除物料（级联删除关联工序）
- 同名称不同版本显示为独立行

### 4.2 工人管理对话框（树形结构重构）

```
┌──────────────────────────────────┐
│  管理工人                     - □ × │
├──────────────────────────────────┤
│  班组·工人列表      树形展开       │
│  ┌──────────────────────────┐    │
│  │ 📁 装配组                │    │
│  │   ├── 👤 张三            │    │
│  │   ├── 👤 李四            │    │
│  │   └── 👤 王五            │    │
│  │ 📁 测试组                │    │
│  │   ├── 👤 赵六            │    │
│  │   └── 👤 孙七            │    │
│  │ 📁 包装组                │    │
│  │   └── 👤 周八            │    │
│  └──────────────────────────┘    │
│                                    │
│  添加工人区域                      │
│  姓名:[        ] 班组:[ 选择▼ ]   │
│  [添加工人] [编辑工人] [删除工人] │
│                                    │
│  管理班组区域                      │
│  班组名:[        ]                 │
│  [添加班组] [删除班组]             │
└──────────────────────────────────┘
      尺寸: 620×550
```

**交互逻辑**：
- 树形结构：班组为父节点，工人为子节点，可展开/折叠
- 添加班组：输入班组名 → 添加到树作为根节点
- 添加工人：输入姓名 + 选择班组（下拉框列出所有班组）→ 添加到选中班组下
- 编辑工人：选中工人 → 弹窗修改姓名和班组
- 删除工人：选中工人 → 确认删除
- 删除班组：选中班组 → 确认删除（连带删除组内所有工人）
- 班组下拉选项动态更新（添加/删除班组后刷新）

### 4.3 工序管理对话框（重构-去掉工人分配）

```
┌──────────────────────────────────┐
│  管理工序                      - □ × │
├──────────────────────────────────┤
│  工序列表                         │
│  ┌────────────────┬──────┬──────┐│
│  │ 物料           │ 工序  │ 单价  ││
│  ├────────────────┼──────┼──────┤│
│  │TR-001(变压器)  │ 绕线  │ 1.5  ││
│  │CONT-01(接触器) │ 组装  │ 2.0  ││
│  │CB-001(断路器)  │ 测试  │ 2.5  ││
│  │ ...            │ ...  │ ...  ││
│  └────────────────┴──────┴──────┘│
│                                    │
│  物料:[可搜索下拉 ▼     ]         │
│       ╔═══════════════════╗       │
│       ║ TR-001(变压器)    ║       │  ← 输入关键字过滤
│       ║ TR-002(变压器)    ║       │
│       ║ CONT-01(接触器)   ║       │
│       ║ CB-001(断路器)    ║       │
│       ║ RE-001(继电器)    ║       │
│       ║ ...              ║       │
│       ╚═══════════════════╝       │
│  工序:[可搜索下拉 ▼     ]         │
│  单价:[        ]                  │
│                                    │
│  [添加] [编辑] [删除]             │
└──────────────────────────────────┘
      尺寸: 650×450
```

**交互逻辑**：
- 物料下拉：从物料库加载，显示"编号(名称)-版本"格式
  - 输入关键字（如"变压"）自动过滤匹配物料
  - 支持拼音/中文/编号模糊匹配
- 工序下拉：从已有工序名中加载，也可输入新工序名
- 单价：数字输入，支持小数
- 选中表格行后，物料/工序/单价自动填入输入框，方便编辑
- 完全移除"工人分配"区域及相关逻辑

### 4.4 主界面"添加工序"物料显示优化

主界面的添加记录表单中，"工序"下拉选项也显示为"编号(名称)-版本 / 工序名"格式，如：
```
TR-001(变压器)-V1.0 / 绕线
CONT-01(接触器)-A型 / 组装
```

## 6. 假设点

- 物料编号由用户手动输入，不自动生成
- 班组管理不支持层级嵌套（班组只有一级，工人是第二级）
- 删除班组时，该班组下的所有工人也将被删除
- 工序名支持从已有列表中选择或输入新名称
- 物料产量记录（主界面）的工序选择格式为"编号(名称)-版本 / 工序名"
