# 生产管理系统 — 桌面端应用设计文档

> 版本 1.0 | 2026-07

---

## 1. 需求分析

### 1.1 核心功能需求

| 编号 | 功能模块 | 描述 | 优先级 |
|------|---------|------|--------|
| F1 | 用户登录 | 用户名+密码验证，记住密码，角色区分（admin/leader/worker） | P0 |
| F2 | 生产记录管理 | 添加工人工序产量记录，删除记录，按日期/工人筛选 | P0 |
| F3 | 物料管理 | 物料CRUD（名称、单价） | P0 |
| F4 | 工人管理 | 工人CRUD（姓名、组别） | P0 |
| F5 | 工序管理 | 工序CRUD（物料、工序名、单价）+ 工人分配 | P0 |
| F6 | 用户管理 | 用户CRUD（用户名、密码、角色） | P1 |
| F7 | 权限管理 | 按用户按权限项勾选（管理员可见） | P1 |
| F8 | 数据统计 | 汇总查询（按日期/工人筛选），月度汇总 | P0 |
| F9 | 图表报表 | ECharts 图形化展示（工人工资排行、日产量趋势、工序分布） | P0 |
| F10 | Excel导出 | 统计结果导出到 .xlsx 文件 | P1 |
| F11 | 密码修改 | 当前登录用户可修改自己的密码 | P1 |

### 1.2 目标用户群体

| 角色 | 权限范围 | 典型操作 |
|------|---------|---------|
| 管理员 (admin) | 全部权限 | 管理用户、分配权限、查看所有数据 |
| 班组长 (leader) | 除用户管理外的全部功能 | 管理物料/工人/工序、查看统计、导出报表 |
| 普通用户 (worker) | 仅查看本人记录、添加记录 | 查看个人产量、工资 |

### 1.3 使用场景

- **场景A（日常录入）**：班组长打开系统 → 登录 → 选择工人 → 选择工序 → 填写件数 → 添加记录
- **场景B（数据查看）**：管理员登录 → 查看主面板统计数据 → 点击图表按钮 → 浏览器打开图表报告
- **场景C（基础维护）**：管理员打开物料/工人/工序管理 → 添加/修改/删除基础数据
- **场景D（月度结算）**：班组长打开月度汇总 → 选择月份 → 查看各工人产量工资 → 导出Excel

## 4. 视觉风格

### 4.1 色彩方案

| 用途 | 色值 | 名称 | 说明 |
|------|------|------|------|
| 背景色 | `#f0f2f5` | 浅灰蓝 | 主内容区域背景 |
| 卡片/控件背景 | `#ffffff` | 白色 | 表单、对话框、卡片 |
| 主色调 | `#1a73e8` | Google Blue | 按钮、顶栏、链接 |
| 强调色 | `#e67e22` | 橙色 | 添加按钮、重要提示 |
| 成功 | `#27ae60` | 绿色 | 导出、汇总按钮 |
| 危险 | `#e74c3c` | 红色 | 删除按钮、错误提示 |
| 深色文字 | `#2c3e50` | 深蓝灰 | 标题、主体文字 |
| 浅色文字 | `#888888` | 灰色 | 辅助说明文字 |

### 4.2 字体选择

- **主字体**：Microsoft YaHei（微软雅黑）
- **字号**：标题 16px/12px，正文 10/11px，辅助 8/9px
- **按钮**：9px Bold

### 4.3 整体风格

- **风格定位**：**简约商务风（Flat Design）**
- 扁平化按钮，无渐变无阴影
- 圆角边框（通过边框厚度和颜色营造）
- 卡片式分区（白色背景块在灰色背景上）
- 清晰的信息层级和留白

### 4.4 交互反馈

- 按钮 hover 时显示手型光标 (`cursor='hand2'`)
- 错误使用 MessageBox 弹窗
- 成功操作使用 Info 弹窗提示
- 删除操作需二次确认

## 5. 假设点

- 运行环境为 Windows，默认字体使用微软雅黑
- 使用 SQLite 本地数据库，无需网络
- 图表渲染依赖本地 ECharts JS 文件
- Excel 导出依赖 openpyxl 库（按需安装）
